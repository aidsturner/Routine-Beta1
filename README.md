# Routine-Beta1
# Calendar
# It helps people keep track of their events and organize their schedule. It also facilitates communicating with group members. The target audience is TTU students participating in group projects.
# Home - Most of the page is a calendar of the month which adds bullet points on the days that you have events. On top of the page, there is a navigation bar which takes you to the page you clicked on.
# Profile - On top of the page, there is a navigation bar which takes you to the page you clicked on. The rest of the page is all boxes where you fill out your personal information with an option to add a profile picture.
# Events - On top of the page, there is a navigation bar which takes you to the page you clicked on. This page shows specific events you have on the day you selected. It is a timeline of the day with titled boxes showing what time you have your event.
# Social - Shows friends and people that have been added to your calendar, with a navigation bar at the top of the page.
# Interactive feature 1 - On the calendar / home page, there is a calendar with a customizable image on the side / top depending on the device you are viewing it on.
# Interactive feature 2 - Notification bell at the top right of each page to notify you of any newly created events, friend requests, or collaborator invites.
# Interactive feature 3 - On the profile page, there is a profile picture that the user can edit and input their own image.
# Interactive feature 4 - On the events page, there is a fuller view of the present day with times of different created events. You are able to share specific events with others and see theirs as well.
